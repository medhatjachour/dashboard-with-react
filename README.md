## Installation

### Steps
1. Clone the repository:
    ```bash
    git clone https://github.com/medhatjachour/dashboard-with-react.git
    cd dashboard-with-react

    ```


2. Install the required packages:
    ```bash
    npm install
    ```

4. Run the application:
    ```bash
    npm run dev
    ```
